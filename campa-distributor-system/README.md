# Campa Distributor Billing & Delivery System

A cloud-based distributor billing and delivery management system for a Campa beverage distribution business.

## Project Structure

- `backend`: Node.js + Express + PostgreSQL API
- `admin-web`: React + Tailwind CSS Dashboard
- `mobile-app`: Flutter App for Field Staff

## Getting Started

### Backend

1. Navigate to `backend` directory.
2. Run `npm install`.
3. Create a `.env` file based on `.env.example`.
4. Run `npm run dev` to start the server.
